This page demos a page theme with shaded edge on the left and right, and center the content in the middle.

Entire page is enclosed by a table cell, where the page (cell) width is fixed to 960px. A lot of websites set their width at 950-1000px.

Page width of some other websites: mitbbs.com is 950px. 6park.com is 980px. wenxuecity.com is 990px. netease.com is 960px. sohu.com is 950px. sina.com.cn is 1000px. manoa.hawaii.edu is 960px. whitehouse.gov is 980px.

Some websites may additionally use a longer banner (wenxuecity.com) or background image (sohu.com) at the top to spann the entire screen width.

Background color is set to light blue-green (#eeffff), so transparent to show the background image.

Another common web page arrangements is have a main banner on the left side. It's not so common for the banner to be on the right side.
